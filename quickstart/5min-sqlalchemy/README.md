# ⚡ 5-Minute Quickstart: SQLAlchemy + CUBRID

## 1) Prerequisites
Install Docker Desktop (or Docker Engine + Docker Compose).

## 2) Start everything
```bash
docker compose up -d
```

## 3) Verify quickstart is running
```bash
pip install -r requirements.txt
python app.py
```

You should see successful Core and ORM CRUD output.
